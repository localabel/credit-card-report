# 💳 Dashboard Laporan Kartu Kredit
> **Platform Belajar:** MySkill | **Tools:** Google Looker Studio

---

## 📌 Tentang Project Ini
Project ini adalah dashboard analisis data yang dibuat menggunakan **Google Looker Studio** sebagai bagian dari program belajar Data Analyst di MySkill. Dashboard ini menyajikan analisis lengkap data pelanggan kartu kredit, mulai dari demografi pelanggan, perilaku transaksi, pendapatan, hingga profil risiko.

---

## 🎯 Tujuan Analisis
- Memahami profil demografi pelanggan kartu kredit
- Menganalisis volume transaksi dan pola pendapatan berdasarkan segmen pelanggan
- Mengidentifikasi segmen pelanggan berisiko tinggi berdasarkan data akun menunggak
- Mendukung pengambilan keputusan berbasis data untuk strategi bisnis kartu kredit

---

## 📊 Ringkasan Dashboard

### 👥 Demografi Pelanggan
- Total pelanggan: **5.054**
- Total transaksi: **326.544**
- Rata-rata skor kepuasan: **2,25**
- Total nilai transaksi: **$22.314.344,00**
- Pelanggan dikelompokkan berdasarkan **Negara Bagian, Pekerjaan, Tingkat Pendidikan, dan Status Pernikahan**

### 💰 Transaksi & Pendapatan
- **Pengusaha (Businessman)** mencatatkan bunga dan biaya tahunan tertinggi
- Analisis Revenue Generator berdasarkan **Tingkat Pendidikan x Pekerjaan**
- Volume transaksi dipantau per kelompok pekerjaan

### ⚠️ Profil Risiko Pelanggan
- Akun menunggak dianalisis berdasarkan **Negara Bagian dan Segmen**
- **Oregon (OR)** mencatatkan tingkat akun menunggak tertinggi sebesar **33,33%**
- Rincian risiko tersedia berdasarkan tingkat pendidikan dan jenis pekerjaan
